# Analisi Attrition HR

## Obiettivo
Quali fattori (stipendio, straordinari, distanza casa-lavoro, soddisfazione) predicono meglio l'abbandono dei dipendenti?

## Dataset
IBM HR Analytics Employee Attrition (Kaggle)
1470 dipendenti, 35 variabili.

## Stack
Python (pandas, scipy, scikit-learn), SQLite, Tableau Public
